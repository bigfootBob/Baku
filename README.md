# Baku — Eater of Nightmares

> *"I am the eater of nightmares. I hunger for your worries."*

**Baku** is a web app (iOS, Android maybe in the future) built on the Japanese myth of the Baku in an effort to give users a way to digitally release their concerns (digital therapy). Available online at: [https://silvanian.art/](https://silvanian.art/).

The Baku is a spirit that devours nightmares. Users type a worry or burden, feed it to the Baku, and receive a short, poetic response generated by Google Gemini AI. Each feeding earns XP and levels up your Baku over time.

This project is a companion app for the [Uncanny Coffee Hour Podcast](https://www.uncannycoffeepodcast.com/).

## Features

- **Feed your worries** — Type a thought, press Feed, and the Baku devours it with a short, poetic AI-generated response
- **Animated Baku character** — The Baku transitions between sleeping, waking, eating, processing, and wise states with smooth crossfade animations and a pulsing border while thinking
- **XP & leveling system** — Earn 20 XP per worry fed; every 100 XP levels up your Baku with a fireworks celebration modal
- **Onboarding flow** — A 3-step illustrated introduction to the Baku on first launch
- **Crisis-aware responses** — If a message indicates self-harm or crisis, the backend returns only the 988 Suicide & Crisis Lifeline
- **Bot protection** — Honeypot field on web builds silently rejects automated submissions
- **Firebase App Check** — Prevents unauthorized API access from non-app clients
- **Dark & light mode** — Respects the user's system preference via `userInterfaceStyle: "automatic"`

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18+
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- A Firebase project with **Authentication**, **Cloud Functions**, and **App Check** enabled
- A [Google Gemini API key](https://aistudio.google.com/)

### 1. Install dependencies

```bash
npm install
cd functions && npm install && cd ..
```

### 2. Configure Firebase

Create `data/firebaseConfig.ts` with your Firebase project credentials:

```ts
import { initializeApp } from 'firebase/app';

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  // ...
};

export const app = initializeApp(firebaseConfig);
```

### 3. Configure the Cloud Function

Create a `.env` file in the `functions/` directory:

```
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Run the Firebase emulator (for local development)

```bash
firebase emulators:start --only functions
```

### 5. Start the app

```bash
npx expo start
```

Then open in the [Expo Go](https://expo.dev/go) app, an iOS simulator, Android emulator, or your browser.

## Deployment

The app is configured for [EAS Build](https://docs.expo.dev/build/introduction/):

```bash
# Build for iOS
eas build --platform ios

# Build for Android
eas build --platform android
```

Deploy Cloud Functions:

```bash
firebase deploy --only functions
```

## Baku Character States

The `BakuCharacter` component implements a simple state machine:

| State | Trigger | Visual |
|---|---|---|
| `SLEEPING` | App idle, no input | Sleeping Baku image |
| `WAKING` | User starts typing | Sleeping Baku image (transitioning) |
| `EATING` | Feed button pressed | Eating Baku image |
| `PROCESSING` | Awaiting Gemini response | Eating Baku image + pulsing border |
| `IDLE` | Response received | Wise Baku image |

## Crisis Safety

If the Baku detects language indicating a mental health crisis or self-harm, the Cloud Function bypasses the AI response entirely and returns:

> *"Your burden is heavy. Please seek a guide in the waking world who can help you carry it."*
> 988 Suicide & Crisis Lifeline
